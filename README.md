# 🚀 Project Name

A premium, mobile-responsive video membership site designed for high-end digital courses.

## ✨ Key Features
* **Sleek UI/UX:** Clean, modern interface focused on student retention.
* **Dynamic Content:** 33 lessons served through a single-page architecture for easy management.
* **Mobile-First:** Fully responsive design that feels like a native app on iOS and Android.
* **Lightweight:** Zero dependencies. Pure HTML, CSS, and Vanilla JS.
* **Static Hosting:** Optimized for GitHub Pages with HTTPS enabled.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Inter Typography)
- **Logic:** Vanilla JavaScript (Dynamic Lesson Loading)
- **Deployment:** GitHub Pages / Custom Domain
